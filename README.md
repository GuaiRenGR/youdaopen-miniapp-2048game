# youdaopen-miniapp-2048game
一个专为有道词典笔开发的2048游戏miniapp，仅供测试。

## 介绍
此项目主要由 Qwen-3.5-Pro-Thinking 模型开发
已知有道词典笔使用的软件方案为阿里云IoT HaaS UI

## 构建
```
cd 2048-game
nom install
npm run build:prod
```
具体方法请移步[此处](https://www.yuque.com/wcye0k/haasui/gsupny)

## 下载
此miniapp目标设备为A7Pro型号，若有测试需求，你可以在`2048-game`文件夹中下载amr文件测试。由于尚未完善程序，程序尚不能正常运行。待日后会完善程序。
